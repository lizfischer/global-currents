enlarged_capital

Description

Information on Enlarged Capitals.

Attributes

VARCHAR(128) URL NOT NULL The URL of the image.
INT(4) X NOT NULL X coordinate of top left corner of bounding box.
INT(4) Y NOT NULL Y coordinate of top left corner of bounding box.
VARCHAR(4) MS_Number NOT NULL Foreign key, references MS_Number in Folio.
The number of the manuscript.
VARCHAR(8) F_Number NOT NULL Foreign key, references Number in Folio.
The number of the folio or page (e.g. "12 V", "167")
INT(4) W NOT NULL Width of the bounding box.
INT(4) H NOT NULL Height of the bounding box.
VARCHAR(16) Primary_Color The main color of the letter.
For other colors, see EC_Second_Colors
CHAR(1) Letter Which letter was found.
TINYINT(1) IsError NOT NULL 1 if the instance is an error, 0 otherwise.
VARCHAR(128) Notes Notes on image left during tagging.

date

Description

The approximate dates for manuscripts.

Attributes

AUTO_INCREMENT ID NOT NULL Unique identifier with no meanout outside database (surrogate key). Allows for primary key if Super is allowed to be NULL.
TINYINT(2) Century NOT NULL Which century (e.g. 11, 12).
VARCHAR(4) Super Superscript indicating which part of century.
Values: "in" (early), "1" (mid first half), "med" (mid), "2" (mid second half), "ex" (late), NULL (unspecified).
VARCHAR(4) MS_Number NOT NULL The number of the manuscript.
Foreign key, references Number in Manuscript.
Cascade on delete and update.

Keys

  • (ID) {primary}
  • (Century, Super, MS_Number)
